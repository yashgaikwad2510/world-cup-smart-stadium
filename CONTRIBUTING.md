# Contributing to MetLife Assist

First off, thank you for considering contributing to MetLife Assist! It's people like you that make MetLife Assist such a great tool for the FIFA World Cup 2026.

## Developer Guidelines

1. **Linting and Formatting**: Ensure all new code adheres to the project's formatting standards. Run `npm run lint` and `npm run format` before opening a pull request.
2. **Testing**: All new features must include unit tests. Run `npm test` to ensure your code does not break the build. Our CI pipeline enforces a 100% pass rate.
3. **Pull Requests**: Provide a clear description of the problem your PR solves or the feature it adds. Link any relevant issues.

## Development Setup

```bash
npm install
npm run dev
```
